================================================================
  FLUJO DE LA APLICACIÓN
================================================================

1. El navegador carga index.html
2. index.html ejecuta main.jsx
3. main.jsx monta <App /> dentro del <BrowserRouter>
4. App.jsx define las rutas y el estado global
5. Según la URL, React Router muestra la página correspondiente
6. Cada página usa componentes para construir su interfaz


================================================================
  ARCHIVO: main.jsx
================================================================

Es el archivo de arranque. Solo hace tres cosas:
  - Importa Bootstrap CSS y JS (para estilos y carrusel)
  - Importa Bootstrap Icons (para los íconos bi-scissors, etc.)
  - Importa styles.css (estilos propios del proyecto)
  - Monta <App /> dentro de <BrowserRouter> en el div#root del HTML

Sin este archivo la app no arrancaría.


================================================================
  ARCHIVO: App.jsx
================================================================

Es el componente raíz que contiene toda la app.

ESTADO QUE MANEJA:
  - carrito: arreglo de productos agregados por el usuario
    Se carga desde localStorage al iniciar (useEffect)
    Se guarda en localStorage cada vez que cambia (useEffect)

FUNCIONES:
  - agregarAlCarrito(producto)
      Recibe un producto y lo agrega al carreglo carrito.
      Verifica primero con .find() que no esté duplicado.

  - quitarDelCarrito(id)
      Filtra el carrito con .filter() para eliminar el producto
      cuyo id coincida con el recibido.

RUTAS DEFINIDAS:
  /             → Inicio.jsx
  /servicios    → Servicios.jsx
  /reserva      → Reserva.jsx
  /productos    → Productos.jsx  (recibe agregarAlCarrito y carrito)
  /carrito      → Carrito.jsx    (recibe carrito y quitarDelCarrito)
  *             → NoEncontrado.jsx  (cualquier ruta desconocida)

COMPONENTES SIEMPRE VISIBLES:
  <BarraNavegacion> aparece en todas las páginas (arriba)
  <PiePagina>       aparece en todas las páginas (abajo)


================================================================
  DATOS: servicios.js
================================================================

Exporta un arreglo llamado "servicios" con 6 objetos.
Cada servicio tiene:
  id        → identificador único
  nombre    → nombre del servicio
  categoria → Corte / Combo / Afeitado / Color / Tratamiento / VIP
  precio    → precio en soles (S/)
  duracion  → tiempo estimado (ej: "45 min")
  descripcion → texto descriptivo
  icono     → clase de Bootstrap Icons (ej: "bi-scissors")
  imagen    → ruta de la imagen (ej: "/assets/corte-clasico.jpg")

Este arreglo lo usan: Inicio, Servicios, FormularioReserva.


================================================================
  DATOS: productos.js
================================================================

Exporta un arreglo llamado "productos" con 6 objetos.
Cada producto tiene:
  id        → identificador único
  nombre    → nombre del producto
  categoria → Cabello / Barba / Shampoo
  precio    → precio en soles (S/)
  medida    → cantidad/peso (ej: "100 g", "30 ml")
  descripcion → texto descriptivo
  imagen    → ruta de la imagen
  nuevo     → true/false (muestra etiqueta NUEVO)
  stock     → "disponible" o "pocas" (muestra etiqueta EN STOCK / ÚLTIMAS)

Este arreglo lo usan: Productos, TarjetaProducto.

================================================================

Barra de navegación fija en la parte superior (sticky-top).

================================================================
  COMPONENTE: TarjetaServicio.jsx
================================================================

Tarjeta visual para mostrar un servicio individual.

PROPS QUE RECIBE:
  servicio → objeto con los datos del servicio
  numero   → número de orden (01, 02, 03...)

FUNCIONALIDAD:

================================================================
  COMPONENTE: TarjetaProducto.jsx
================================================================

Tarjeta visual para mostrar un producto individual.

PROPS QUE RECIBE:
  producto        → objeto con los datos del producto
  agregarAlCarrito → función de App.jsx
  carrito         → arreglo actual del carrito

FUNCIONALIDAD:
  - Muestra imagen, etiqueta NUEVO (si aplica), etiqueta de stock
  - Muestra categoría, nombre, descripción, precio, medida
  - Botón AGREGAR:
      Verifica con .some() si el producto ya está en el carrito
      Si ya está: muestra "AGREGADO" y desactiva el botón
      Si no está: llama a agregarAlCarrito(producto)


================================================================
  COMPONENTE: FiltrosServicios.jsx
================================================================

Panel de filtros para la página de Servicios.

PROPS QUE RECIBE:
  categorias            → arreglo de categorías únicas
  categoriaSeleccionada → categoría activa
  setCategoriaSeleccionada → función para cambiarla
  busqueda              → texto del buscador
  setBusqueda           → función para cambiarlo
  precioMaximo          → valor del slider
  setPrecioMaximo       → función para cambiarlo

CONTROLES:
  - Input de texto: filtra servicios por nombre
  - Select de categoría: filtra por categoría
  - Slider de precio: filtra servicios con precio <= precioMaximo

Todos los filtros se aplican juntos (con &&) en Servicios.jsx.


================================================================
  COMPONENTE: FiltrosProductos.jsx
================================================================

Panel de filtros para la página de Productos.

PROPS QUE RECIBE:
  categorias            → arreglo de categorías únicas
  categoriaSeleccionada → categoría activa
  setCategoriaSeleccionada → función para cambiarla
  busqueda              → texto del buscador
  setBusqueda           → función para cambiarlo

CONTROLES:
  - Input de texto: filtra productos por nombre
  - Botones de categoría (con Bootstrap + clase .boton-filtro):
      TODOS + uno por cada categoría existente
      El botón activo se resalta en dorado

Ambos filtros se aplican juntos (con &&) en Productos.jsx.


================================================================
  COMPONENTE: FormularioReserva.jsx
================================================================

Formulario completo para reservar un turno.

ESTADO QUE MANEJA (useState):
  formulario → objeto con todos los campos:
    nombre, telefono, servicio, barbero, fecha, hora, aceptarCondiciones
  mensaje → texto de confirmación o error

FUNCIONALIDAD:
  - Lee location.state para saber si se llegó desde una tarjeta de servicio.
    Si es así, precarga el campo "servicio" con ese dato.
  - manejarCambio(): actualiza cualquier campo del formulario con un solo handler
    usando [name] como clave dinámica
  - seleccionarBarbero(): actualiza el campo barbero al hacer clic en una tarjeta
  - manejarEnvio():
      Valida que se haya elegido barbero
      Valida que se hayan aceptado las condiciones
      Si todo está bien, muestra mensaje de confirmación
      Resetea el formulario después del envío

================================================================
  COMPONENTE: ResumenCarrito.jsx
================================================================

Muestra el contenido del carrito y el formulario de envío.

PROPS QUE RECIBE:
  carrito          → arreglo de productos
  quitarDelCarrito → función para eliminar un producto

ESTADO QUE MANEJA (useState):
  formulario → { nombre, telefono, direccion }
  mensaje    → texto de confirmación del pedido

FUNCIONALIDAD:
  - Si el carrito está vacío: muestra pantalla de carrito vacío con link a Productos
  - Si tiene productos:
      Lista cada producto con imagen, nombre, categoría, medida y precio
      Cada producto tiene botón de eliminar (llama a quitarDelCarrito)
      Calcula el total con .reduce() sumando todos los precios
      Formulario de datos de envío: nombre, teléfono, dirección
      Al confirmar muestra mensaje de éxito y resetea el formulario

================================================================
  FLUJO 1: RESERVAR UN SERVICIO
================================================================

1. Usuario entra a /servicios
2. Ve las tarjetas de servicios
3. Hace clic en RESERVAR de una tarjeta
   → TarjetaServicio usa navigate('/reserva', { state: { servicio } })
4. Llega a /reserva con el servicio precargado en el formulario
5. Elige barbero, fecha y hora
6. Acepta condiciones y hace clic en CONFIRMAR RESERVA
7. FormularioReserva valida y muestra mensaje de éxito


================================================================
  FLUJO 2: COMPRAR UN PRODUCTO
================================================================

1. Usuario entra a /productos
2. Puede filtrar por categoría o buscar por nombre
3. Hace clic en AGREGAR en una tarjeta
   → llama a agregarAlCarrito(producto) en App.jsx
   → el producto se agrega al estado carrito
   → el botón cambia a "AGREGADO" y se desactiva
   → el contador en la barra de navegación aumenta
4. Usuario va a /carrito
5. Ve la lista de productos y el total calculado con .reduce()
6. Puede quitar productos con el botón de basura
   → llama a quitarDelCarrito(id) en App.jsx
7. Completa nombre, teléfono y dirección
8. Hace clic en CONFIRMAR PEDIDO → mensaje de éxito

NOTA: El carrito se guarda en localStorage automáticamente.
Si el usuario cierra y vuelve a abrir el navegador,
el carrito sigue ahí gracias al useEffect en App.jsx.
